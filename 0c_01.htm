紐約國際之服務階段方式

A .規劃顧問階段：
  ( a ) 硬體規劃部份
    1.提出分析與建議：
      (a) 整體設施之配置坪數與動線。
      (b) 各空間所需硬體設備項目、數量與等級，如爐具、健身器
        等。
    2. 與室內設計師配合規劃空間配置。
    3. 於施工期間與營建單位配合確保設施之功能。
( a ) 硬體規劃部份
  ( b ) 財務部份
    1.提出整個公設俱樂部所需之硬體設備的項目與預估費用表。
    2.評估各項活動設備及器材。
    3.營運後每個月之預估總損益表。
    4.籌備營運期間，有關人事行政之編製與開銷等財務預算。
( c ) 人事
    依所呈報之人事編制，於開幕前招募而且訓練完成，同時編列合乎
    本案之人事規章及人事訓練。
( d ) 行銷計劃
      1. 依市場需求，規劃可行性之社區會員各項收費辦法。
      2. 休閒俱樂部之餐飲內容收費及其他設備收費辦法。
      3. 對外宣傳及媒體公關之運作策略。
 

B .顧問管理階段：
以專業能力來實際經營管理俱樂部或公共設施。其配合方式有多種，例如：
    ( a ) 純為顧問，不介入經營管理。
    ( b ) 介入實際經營管理。
